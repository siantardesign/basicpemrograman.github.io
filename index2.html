Basic Pemrograman Web - Html

1. Pengenalan
Buka File Explorer dan buat folder baru
2. Isi Folder
Buat folder dengan menekan tombol "new" dipojok kiri atas
3. Aplikasi
Setelah membuat folder, buka aplikasi kode editor seperti Visual Studio Code. Klik "File" pada bagian atas kiri dan klik "Open Folder". Jika sudah, pilih folder yang sudah disiapkan
4. HTML 5
ketik "html:5" untuk rangka dasar
5. Header, Paragraf, dan Styling
Header adalah judul, sementara Paragraf adalah isi. B untuk teks tebal, I untuk teks miring, dan U untuk menggaris bawahi teks
6. Tabel
TR adalah baris tabel, sedangkan TR isi tabelnya
7. Link
Sebagai tautan
8. List
OL untuk list bernomor, UL untuk list tanpa nomor
9. BR
BR (break) untuk memisahkan antara dua atribut
